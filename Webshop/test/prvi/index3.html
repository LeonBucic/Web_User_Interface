Internet i CSS primjer

Internet je globalni sustav međusobno povezanih računalnih mreža koje koriste standardni protokol komunikacije. Internet je postao ključna tehnologija u svim aspektima našeg života, uključujući komunikaciju, obrazovanje, trgovinu i zabavu.

DIV element obično koristimo kada želimo grupirati elemente, primjerice za stvaranje nekog bloka sadržaja, a kasnije stilizirati taj blok pomoću CSS-a. Možemo dodati klasu, ID ili drugi atribut divu kako bismo ga mogli ciljati i stilizirati prema potrebi.

CSS (Cascading Style Sheets)

CSS ima svojstva poput boje pozadine, veličine fonta, margina i razmaka između elemenata. Ove stilove primjenjuje na određene HTML elemente ili grupe elemenata koristeći seletkore. CSS također omogućuje korisnicima da lako mijenjaju izgled web stranica, bez utjecaja na njen sadržaj ili strukturu. Na primjer, promjena boje pozadine cijele web stranice može se vrlo jednostavno izvršiti samo promjenom jednog CSS stilskog pravila. CSS se stalno razvija i poboljšava kako bi se web dizajnerima omogućila još veća kontrola nad izgledom web stranica.
